# v0.1 - Sappiens Framework

Apresentação

O Sappiens Framework é uma aplicação RAD (Rapid Application Development) de código-fonte livre, escrita em PHP, com suporte nativo aos SGBD´s MySQL, Oracle, SQL Server e PostgreSQL.
